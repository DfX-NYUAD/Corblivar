void CorblivarDie::performPacking ( Direction const &  dir)

layout generation: packing, to be performed as post-placement operation

note that packing may undermine alignment requests; to avoid this, we call FloorPlanner::evaluateAlignments before packing but after layout generation (FloorPlanner::evaluateAlignments does annotate alignment success / failure to the blocks themselves, via call to CorblivarAlignmentReq::evaluate)
