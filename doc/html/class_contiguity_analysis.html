Member Function Documentation

void ContiguityAnalysis::analyseBlocks ( int  layers,
std::vector< Block > const &  blocks 
)

Extract blocks' boundaries, and order them by coordinates; this will reduce required comparisons between (in principal all pairs of) blocks notably by considering only relevant blocks. For intra-die contiguity, these are abutting boundaries, and for inter-die contiguity, these are boundaries within a block's outline.
